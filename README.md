# 환자관리프로그램 개발

## 프로젝트 개요

- 프로젝트 내용 : 환자관리 프로그램 개발
- 사용기술 : Qt, C++, SQLite
- 상세 역할 : 프로젝트에 개발계획서 정의서 작성, UI 개발 및 Create, Read, Update, Delete 및 저장, 조건 검색, DB파일 불러오기, csv 내보내기 등의 기능 개발
